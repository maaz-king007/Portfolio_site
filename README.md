## 💻 Running the Portfolio Locally

If you'd like to spin up this React portfolio locally on your machine:

1. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

2. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```
